# 변경 이력 (Changelog)

니케 유레 자동 동기화 도구의 변경 사항을 기록합니다.
연동 유저스크립트(Tampermonkey)와 웹 도구(SPA)를 함께 다룹니다.

## 2026-06-01 — 웹 도구 (member_id 저장 방식 변경)

> 유저스크립트 버전 변경 없음. 웹 도구는 새로고침 시 자동 반영됩니다.

### 변경 — member_id 를 별도 숨김 탭으로 분리
- 기존엔 `유니온 멤버` 탭 B열에 member_id 숨김 열을 삽입했는데, 이로 인해 시트 열이 한 칸씩 밀려 **일부 Apps Script·수식이 멤버 레벨을 잘못 읽는 문제**가 있었습니다.
- 이제 member_id 를 별도 **`_nra_member_mapping`** 숨김 탭에 보관하고, `유니온 멤버` 탭은 **원본 레이아웃**(가입 순서 · 닉네임 · 회차)을 그대로 유지합니다.
- **이미 B열에 member_id 가 삽입된 시트는 자동 복구**됩니다 — 다음 실행 시 그 열을 제거하고 member_id 를 새 탭으로 옮깁니다 (쓰기 전 백업 탭이 먼저 생성됩니다).

## 2026-06-01 — 유저스크립트 v2.4.5

> **Tampermonkey 유저 스크립트를 v2.4.5로 업그레이드** 해야합니다. 웹 도구는 새로고침 시 자동 반영됩니다.

### 신규 기능 — 유저스크립트 버전 감지
- 유저스크립트가 도구로 보내는 데이터에 자신의 버전(`scriptVersion`)을 함께 전달합니다.
- 웹 도구가 설치된 유저스크립트 버전을 감지하여, 권장 버전(현재 v2.4.5)이 아니면 **업그레이드 권장 배너**를 표시합니다 (Greasyfork 링크 포함).
- v2.4.4 이하(버전 미보고)는 "구버전"으로 안내됩니다. 데이터 가져오기를 한 번 실행하면 감지됩니다.

## 2026-06-01 — 유저스크립트 v2.4.4

> **Tampermonkey 유저 스크립트를 v2.4.4로 업그레이드** 해야합니다. 웹 도구는 새로고침 시 자동 반영됩니다.

### 버그 수정
- **빈 시트에서 가용 전체 회차가 백필되지 않던 문제 해결** — 회차 데이터가 하나도 없는 시트(멤버 닉네임만 입력된 새 시트)에서 "신규 회차 데이터 가져오기"를 실행하면 최신 회차 1개만 가져오던 문제를 수정했습니다. 이제 블라블라링크가 제공하는 가용 회차 전체를 한 번에 백필합니다.
- **(웹 도구) 빈 레이드 통계 탭에서 행 삽입 위치 정정** — 회차 데이터가 없는 시트에서 `OO차 레이드 통계` placeholder 행을 밀어내고 그 자리(2행)에 데이터가 들어가던 문제를 수정했습니다. 이제 placeholder 행을 보존하고 그 아래부터 행을 추가합니다.
- **(웹 도구) 탈퇴자 레벨 기록 탭이 갱신되지 않던 문제 해결** — `탈퇴자 레벨 기록` 탭의 grid 행/열이 부족할 때 쓰기가 조용히 실패하던 문제를 수정했습니다. 기록 전 grid를 자동 확장하여 탈퇴자 여러 명도 정상 기록됩니다.

## 2026-06-01 — 유저스크립트 v2.4.3

> **Tampermonkey 유저 스크립트를 v2.4.3로 업그레이드** 해야합니다. 웹 도구는 새로고침 시 자동 반영됩니다.

### 주요 기능 — 다회차 자동 백필
- 시트에 빠진 과거 회차를 자동 감지하여 한 번에 보충 입력합니다.
- **회차 당시 싱크로 레벨**(그 회차에 출전한 니케 최고 레벨)을 회차별로 기록합니다. 현재(최신) 회차에 미참여한 멤버는 현재 싱크로 레벨로, 과거 미참여 회차는 빈 칸으로 둡니다.
- 레이드 통계 · 유니온 멤버 · 레이드 결과 탭을 **각각 따로 판정**하여, 한쪽에만 빠진 회차도 정확히 채웁니다.
- 행·열을 맨 끝이 아니라 **차수 번호 순서에 맞는 위치**에 삽입합니다.
- 적용 전 **"회차별 상세" 미리보기**로 회차마다 채워질 탭·보스를 항목별로 확인할 수 있습니다.
- 적용 후 **최신 회차 미참여 멤버**를 표시합니다.

### 신규 기능 — 탈퇴자 레벨 기록 (해당 탭이 있는 시트 한정)
- 시트에 **`탈퇴자 레벨 기록`** 탭이 있으면, 유니온을 떠난 멤버의 회차별 레벨을 그 탭에 보존합니다 (탈퇴 후 레벨은 기록 안 함).
  - **과거 백필 시**: 과거 레이드에 있으나 현재 유니온에 없는 멤버를 자동 기록 (이미 기록된 회차는 보존).
  - **현재 회차 탈퇴자**: 유니온 멤버 탭에서 빠지기 전, 쌓인 레벨을 탈퇴자 탭으로 이전.
- **탭이 없는 시트는 기존과 동일하게 동작** (탈퇴자 기록 안 함 — 하위호환).

### 버그 수정
- **백업 탭 때문에 적용이 중단되던 문제 해결** — 백업 탭이 회차와 무관하게 **매 실행마다 생성 시각 기반 이름**(`_backup_YYYYMMDD-HHmmss`)으로 새로 만들어져, 같은 회차 재실행·한국어 환경에서도 멈추지 않습니다. 백업은 **최근 3개만 유지**됩니다.
