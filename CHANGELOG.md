# 변경 이력 (Changelog)

니케 유레 자동 동기화 도구의 변경 사항을 기록합니다.
연동 유저스크립트(Tampermonkey)와 웹 도구(SPA)를 함께 다룹니다.

## 2026-06-03 — 웹 도구 (첫 실행 멤버 자동 채움)

> 유저스크립트 버전 변경 없음. 웹 도구는 새로고침 시 자동 반영됩니다.

### 신규 기능 — 유니온 멤버를 미리 입력하지 않아도 됩니다

- `유니온 멤버` 탭이 **완전히 비어 있는 상태**로 [신규 회차 데이터 가져오기]를 누르면, 도구가 수집한 멤버 닉네임을 시트에 **자동으로 채워** 넣고 곧바로 회차 데이터까지 한 번에 처리합니다. 예전처럼 32명 닉네임을 일일이 타이핑하거나, 채운 뒤 다시 [가져오기]를 누를 필요가 없습니다.
- ⚠️ **자동으로 채워진 닉네임은 가입 순서가 아닙니다.** 도구는 가입 순서를 알 수 없어 수집된 순서 그대로 넣습니다. 가입 순서가 중요하면 `유니온 멤버` 탭에서 **직접 정렬**해주세요.
- 닉네임이 **이미 입력된 시트에서는 자동 채움이 동작하지 않아**, 기존에 입력해 둔 데이터를 덮어쓰지 않습니다.

## 2026-06-03 — 유저스크립트 v2.5.0 + 웹 도구 (미리보기 가독성 · 수집 진행 표시)

> **Tampermonkey 유저 스크립트를 v2.5.0으로 업그레이드** 하면 수집 진행 표시가 켜집니다. 웹 도구는 새로고침 시 자동 반영됩니다. (구버전 유저스크립트로도 데이터 가져오기는 정상 동작하며, 진행 표시만 나타나지 않습니다.)

### 개선 — 변경사항 미리보기가 한눈에 들어옵니다
- "예상되는 변경사항"에서 탈퇴·신규·닉네임 변경 멤버에 **시트 행 번호가 함께 표시**됩니다. 같은 닉네임이 여러 명이어도(예: `탈퇴자1`이 두 줄) 어느 행이 바뀌는지 정확히 구분됩니다.
- 추가(＋)·삭제(－)·변경(～)이 **색상과 기호로 구분**되어, 누가 빠지고 누가 들어오는지 한눈에 파악됩니다.

### 신규 기능 — 데이터 수집 진행이 도구 화면에 표시됩니다
- [신규 회차 데이터 가져오기]를 누르면 블라블라링크 새 탭에서 데이터를 모으는 동안, **도구 화면에 수집 진행 상태**가 표시됩니다. 새 탭으로 전환하지 않고도 진행 중임을 확인할 수 있어, 멈춘 줄 알고 다시 누르는 일이 줄어듭니다.

## 2026-06-02 — 유저스크립트 v2.4.6

> 유저스크립트는 Tampermonkey를 통해 v2.4.6으로 자동 업데이트됩니다. 웹 도구는 새로고침 시 자동 반영됩니다.

### 개선 — 새 니케 자동 반영
- 새로 출시되는 니케의 이름이 **자동으로 도구에 반영**됩니다. 신규 니케가 추가돼도 별도 업데이트 작업 없이, 레이드 결과에 출진 니케 이름이 정상 표시됩니다.
- 이 자동 갱신은 조용히 이뤄지며, 그 과정에서 **불필요한 "유저스크립트 업그레이드 권장" 안내가 뜨지 않습니다** (실제로 업그레이드가 필요할 때만 표시됩니다).
- 블라블라링크 화면에 뜨던 "신규 니케 N명" 안내 표시는 제거되었습니다 (이제 자동 반영되어 불필요합니다).

## 2026-06-01 — 웹 도구 (멤버 추적 개선)

> 유저스크립트 버전 변경 없음. 웹 도구는 새로고침 시 자동 반영됩니다.

### 개선 — 닉네임이 바뀌어도 같은 멤버로 추적
- 멤버의 **닉네임이 바뀌어도 같은 사람으로 인식**해, 시트의 닉네임을 자동으로 갱신합니다. 탈퇴·신규 멤버도 자동으로 정리됩니다.
- 일부 시트에서 **멤버 레벨이 한 칸씩 밀려 잘못 읽히던 문제**를 해결했습니다. 이미 영향받은 시트는 다음 실행 때 **자동으로 복구**됩니다 (복구 전 백업 탭이 먼저 생성됩니다).

## 2026-06-01 — 유저스크립트 v2.4.5

> **Tampermonkey 유저 스크립트를 v2.4.5로 업그레이드** 해야합니다. 웹 도구는 새로고침 시 자동 반영됩니다.

### 신규 기능 — 구버전 유저스크립트 안내
- 설치된 유저스크립트가 권장 버전보다 낮으면, 웹 도구가 **업그레이드 안내**(Greasyfork 링크 포함)를 표시합니다. 데이터를 한 번 가져오면 설치된 버전이 확인됩니다.

## 2026-06-01 — 유저스크립트 v2.4.4

> **Tampermonkey 유저 스크립트를 v2.4.4로 업그레이드** 해야합니다. 웹 도구는 새로고침 시 자동 반영됩니다.

### 버그 수정
- **새 시트에서 과거 회차가 한 번에 채워지지 않던 문제** — 회차 데이터가 없는 새 시트(멤버 닉네임만 입력)에서 데이터를 가져오면 최신 회차 1개만 들어오던 문제를 고쳤습니다. 이제 가능한 과거 회차를 한 번에 모두 채웁니다.
- **(웹 도구) 레이드 통계가 맨 위에 잘못 들어가던 문제** — 비어 있던 레이드 통계 탭에서 새 회차가 엉뚱한 위치에 삽입되던 문제를 고쳤습니다. 이제 차수 순서에 맞는 위치에 추가됩니다.
- **(웹 도구) 탈퇴자 레벨 기록이 저장되지 않던 문제** — 탈퇴자가 여러 명일 때 `탈퇴자 레벨 기록` 탭에 기록이 저장되지 않던 문제를 고쳤습니다.

## 2026-06-01 — 유저스크립트 v2.4.3

> **Tampermonkey 유저 스크립트를 v2.4.3로 업그레이드** 해야합니다. 웹 도구는 새로고침 시 자동 반영됩니다.

### 주요 기능 — 빠진 과거 회차 한 번에 채우기
- 시트에 빠져 있는 과거 회차를 자동으로 감지해 **한 번에 채웁니다**. 세 탭(레이드 통계·유니온 멤버·레이드 결과) 중 한쪽에만 빠진 회차도 정확히 보충합니다.
- 각 회차에 **그 회차 당시의 싱크로 레벨**(그 회차에 출전한 니케 최고 레벨)을 기록합니다. 최신 회차에 미참여한 멤버는 현재 레벨로, 과거 미참여 회차는 빈 칸으로 둡니다.
- 회차가 **차수 순서에 맞는 위치**에 들어갑니다.
- 적용 전 **회차별 미리보기**로 어떤 회차·보스가 채워질지 확인하고, 적용 후 **최신 회차 미참여 멤버**를 보여줍니다.

### 신규 기능 — 탈퇴자 레벨 기록 (`탈퇴자 레벨 기록` 탭이 있는 시트만)
- 시트에 `탈퇴자 레벨 기록` 탭이 있으면, 유니온을 떠난 멤버의 회차별 레벨을 그 탭에 보존합니다 (과거에 발견된 탈퇴자도 자동 기록). 탭이 없으면 기존과 동일하게 동작합니다.

### 버그 수정
- **백업 탭 때문에 적용이 멈추던 문제**를 고쳤습니다. 백업은 **최근 3개만** 보관됩니다.
