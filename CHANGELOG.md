# 변경 이력 (Changelog)

니케 유레 자동 동기화 도구의 변경 사항을 기록합니다.
연동 유저스크립트(Tampermonkey)와 웹 도구(SPA)를 함께 다룹니다.

## 2026-06-01 — 유저스크립트 v2.4.2

> 유저스크립트 사용자는 **v2.4.2로 업데이트(재설치)** 가 필요합니다. 웹 도구는 새로고침 시 자동 반영됩니다.

### 주요 기능 — 다회차 자동 백필
- 시트에 빠진 과거 회차를 자동 감지하여 한 번에 보충 입력합니다.
- **회차 당시 싱크로 레벨**(그 회차에 출전한 니케 최고 레벨)을 회차별로 기록합니다.
- 레이드 통계 · 유니온 멤버 · 레이드 결과 탭을 **각각 따로 판정**하여, 한쪽에만 빠진 회차도 정확히 채웁니다.
- 행·열을 맨 끝이 아니라 **차수 번호 순서에 맞는 위치**에 삽입합니다.

### 버그 수정
- **미참가 회차 레벨 오기록 수정** — 가입 전이거나 그 회차에 참여하지 않은 멤버의 싱크로 칸이 채워지던 문제를 고쳤습니다. 이제 실제 참여한 회차만 기록하고, 미참여 회차는 빈 칸으로 둡니다.
- **"최신 회차 미참여 멤버" 정확도 수정** — 과거 회차만 보충해도 항상 진짜 최신 회차 기준으로 표시됩니다. *(유저스크립트 v2.4.2 필요)*
- **백업 탭 생성 실패로 적용이 중단되던 문제 수정 (한국어 환경)** — 같은 회차로 다시 실행해도 백업 탭이 자동으로 다른 이름(`_2`, `_3`…)으로 생성되어 멈추지 않습니다.

### 개선
- **"예상되는 변경사항 → 회차별 상세"** 가독성 개선 — 업데이트되는 탭과 보스 목록을 항목별로 분리.
- 미참여자가 없을 때 문구를 **"최신 회차 모든 멤버 참여 완료"** 로 명확화.
